# 🎮 Soon Soon Link

👉 Demo: https://nickhuang1121.github.io/soon-soon-link/

一款以「快速連線、即時反應」為核心的輕量休閒手遊。  
玩家需要在有限時間內，快速使用手指或滑鼠，一次連接所有淡色方塊來通關，考驗反應速度與視覺判斷能力。

---

## 💡 專案起源

此遊戲靈感來自於在日本富士急樂園體驗的密室逃脫遊戲。  
在遊戲過程中接觸到類似的連線機制，覺得玩法直覺且具有節奏感，因此嘗試將這個概念轉化為數位版本，並重新設計為適合短時間遊玩的輕量手遊。

👉 從實體互動遊戲 → 數位化 → 可重複遊玩的設計轉換

---

## 🧠 遊戲核心概念

- 類似經典「連連看」玩法的延伸版本
- 強調：
  - 速度感（Fast-paced）
  - 直覺操作（Intuitive interaction）
  - 短時間高回饋（Instant feedback loop）

適合碎片時間遊玩（通勤 / 等待空檔）

---

### 🧩 視覺清晰優先
- 簡化 UI，降低辨識成本
- 專注在「找相同」這件事本身

### 🎮 輕量但可重複遊玩
- 單局時間短
- 高重玩性（Replay Value）

---

## 🛠 技術實作

### Frontend
- HTML / CSS / JavaScript（Vanilla JS）

### Game Logic
- 狀態管理（Board State）
- 配對判斷邏輯
- 時間與分數系統

### 架構設計重點
- 遊戲邏輯與 UI 分離
- Function responsibility 明確
- 可擴充設計（關卡 / 難度）

---

## 📐 系統設計思路

本專案刻意採用「簡單技術 + 清楚架構」：

- 不依賴大型框架
- 專注在：
  - 邏輯清晰度
  - 可讀性
  - 可維護性
